# 🚀 75 Days LeetCode Challenge

Welcome to my **#75DaysLeetCodeChallenge** repository!

This challenge is my commitment to solving **Data Structures and Algorithms problems on LeetCode for 75 consecutive days**. The aim is to strengthen problem-solving skills, improve coding consistency, and deepen my understanding of core DSA concepts.

---

## 📌 Challenge Overview

- **Duration:** 75 Days  
- **Platform:** LeetCode  
- **Language:** Java  
- **Focus:** Data Structures & Algorithms  
- **Goal:** Improve problem-solving skills and prepare for technical interviews.

Every day I solve at least one problem covering various DSA topics and push the solution to this repository.

---


## 🧠 Topics Covered

During this challenge, I aim to practice problems from multiple DSA topics including:

- Arrays  
- Strings  
- Hashing  
- Two Pointers  
- Sliding Window  
- Recursion  
- Linked Lists  
- Stacks & Queues  
- Trees  
- Graphs  
- Backtracking  
- Dynamic Programming  
- Greedy Algorithms  

---

## 🎯 Purpose of This Challenge

The objective of this challenge is to:

- Build a strong foundation in **Data Structures & Algorithms**
- Develop **consistent coding habits**
- Improve **logical thinking and optimization skills**
- Prepare for **technical interviews and coding assessments**
